Set up real email sending

    
OneMan sends real email through your own free EmailJS account. Create one at emailjs.com, connect an email service, build a template, then paste the three IDs below. Stored only in this browser.

    

    

    

    

    
    

    
Important — so OneMan’s email is what actually sends (not the template’s own text):
In your EmailJS template, delete any sample/hard-coded copy and set the fields to these EXACT variables:
• Subject = {{subject}}
• Content / body = {{message}} — switch the content box to the code/HTML view and paste only this, so it is not escaped
• To Email = {{to_email}}
• From Name = {{from_name}}   Reply To = {{reply_to}}
If the body contains any fixed text instead of {{message}}, recipients get the template, not OneMan’s email.
